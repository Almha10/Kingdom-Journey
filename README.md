# Kingdom Journey 🇸🇦

An Arabic interactive travel web app that showcases regions, landmarks, and achievements across Saudi Arabia through a modern visual experience.

## 🌐 Live Demo
[View Website](https://kingdom-journey--almha1.replit.app/)

## 📌 Project Overview
Kingdom Journey is a web application designed to present Saudi Arabia’s regions in an engaging and interactive way.  
The project highlights places such as Riyadh, Jeddah, AlUla, Tabuk, Aseer, and Al-Qassim with a modern Arabic interface.

## ✨ Features
- Arabic RTL interface
- Interactive region cards
- Modern dark visual design
- Smooth navigation between sections
- Region-based exploration experience
- Responsive web layout

## 🛠 Technologies Used
- React / Vite
- JavaScript
- HTML
- CSS
- Replit

## 🎯 Purpose
This project was created to showcase Saudi regions in a digital, interactive, and visually engaging experience.
